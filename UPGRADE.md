# 升级帮助文件

## v0.2-alpha ~
1. 新建数据库`wocenter_advanced`
2. 执行以下命令：
```bash
$ git clone https://github.com/<username>/wocenter_advanced.git #重新克隆项目
$ cd wocenter_advanced #进入项目根目录
$ composer install -v
$ ./yii wocenter/install
```

## v0.1.5
